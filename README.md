# 📈 Stock Portfolio Tracker

A Python-based stock portfolio management application that allows users to track investments using predefined stock prices.

This project demonstrates Python fundamentals including dictionaries, file handling, arithmetic operations, object-oriented programming, and user input validation.

---

# Features

- Add stocks and quantities
- Hardcoded stock price database
- Calculate total portfolio value
- Portfolio summary table
- Export report to TXT file
- Export report to CSV file
- Input validation
- Clean OOP architecture

---

# Technologies Used

- Python 3.x
- CSV Module
- Datetime Module

No external libraries required.
